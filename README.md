# Smart Attendance System

A digital solution that automatically records and manages student attendance in real time. It helps teachers and parents track presence, absences, and important updates efficiently through notifications and an easy-to-use interface.

## Live Demo
🚀 **[https://smart-attendance-system-ashen-iota.vercel.app](https://smart-attendance-system-ashen-iota.vercel.app)**

## Features
- **Real-time Attendance Tracking**: Keep track of student attendance seamlessly.
- **Auth0 Authentication**: Secure login and signup system.
- **Dark & Light Mode**: Beautiful glassmorphism UI with full theme support.
- **Teacher-Principal Chat**: Integrated communication channel for staff.
- **Alert System**: Notifications for absent students and upcoming exams.
